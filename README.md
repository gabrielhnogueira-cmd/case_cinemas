# CASE CINEMAS

Sistema desenvolvido para a disciplina de Engenharia de Software com o objetivo de aplicar conceitos de UML, arquitetura em camadas e persistência de dados utilizando Python e SQLite.

---

# Funcionalidades

- Cadastro de filmes
- Listagem de filmes

---

# Diagramas UML

O projeto contém os seguintes diagramas:

- Diagrama de Casos de Uso
- Diagrama de Classes
- Diagrama de Atividade
- Diagrama de Sequência
