# Levantamento de Requisitos – CASE CINEMAS

## Objetivo do Sistema

O sistema CASE CINEMAS tem como objetivo auxiliar no gerenciamento de filmes e sessões de cinema, permitindo o cadastro e visualização de informações de forma organizada.

---

# Requisitos Funcionais

RF01 – O sistema deve permitir o cadastro de filmes.

RF02 – O sistema deve permitir listar os filmes cadastrados.

RF03 – O sistema deve permitir visualizar sessões disponíveis.

RF04 – O sistema deve permitir a compra de ingressos.

RF05 – O administrador deve gerenciar os filmes do sistema.

---

# Regras de Negócio

RN01 – Todo filme deve possuir título, gênero, duração e classificação.

RN02 – A duração do filme deve ser maior que zero.

RN03 – Não é permitido cadastrar filmes sem título.

RN04 – O ingresso deve estar vinculado a uma sessão.

RN05 – Apenas administradores podem cadastrar filmes.

---

# Usuários do Sistema

## Administrador
Responsável pelo gerenciamento de filmes e sessões.

## Espectador
Responsável por visualizar sessões e comprar ingressos.
